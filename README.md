# JSP_Mall
3학년 1학기 '웹서버컴퓨팅' 기말 프로젝트

# <소개>
- JSP(JavaServer Pages) 쇼핑몰 프로젝트
- 데이터베이스(MySQL)를 활용하여, 회원, 상품, 재고의 CRUD 구현
- 쿠키를 이용한 장바구니 구현
- 웹서버는 <a href = "Apache Tomcat" >Apache Tomcat</a>을 사용
